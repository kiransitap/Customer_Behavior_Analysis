# Customer_Behavior_Analysis

Aim of this project is to analyze customer shopping behavior using Python, SQL, and Power BI and generate actionable insights to support data-driven business decisions. This project demonstrates an end-to-end data analytics pipeline, showcasing practical skills in data preprocessing, SQL querying, and business intelligence reporting, similar to real-world industry use cases.
